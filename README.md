# Overview

Simple Rock, Paper, Scissors web application that uses a combination of Haversine formula and time between requests to find opponent.

# Technology

## Frontend
- React
- Fetch

## Backend
- Spring Boot 
- Spring MVC (REST)

## Persistence
- Hazelcast In-Memory Data Grid
- Redis

# Build Instructions

## Compile

1) From project root: `mvn clean package`
